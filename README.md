# HKSI Android App - README
A simple school information lookup Android application developed with Android Studio, following university lab exercises (List, Search, WebView, Localization, Pagination).

## Minimum Requirements
- minSdk: 29 (Android 10)
- compileSdk: 36
- Android Studio Hedgehog or later
- Internet connection required

## Features
- Real-time school name search
- Pagination control
- Empty search result hint
- School details dialog
- Built-in WebView for official school website
- Bilingual: English / Traditional Chinese (Hong Kong)
